## **💾 Installation & Usage**  
### **Clone the Repository**  
```bash
git clone https://github.com/your-username/spotify-clone.git
cd spotify-clone
```
### **Install Dependencies**  
```bash
npm install
```
### **Run the App**  
```bash
npm run dev
```
*(Make sure you have Node.js installed!)*

## **🚀 Deployment**  
This project is deployed on **Vercel**. To deploy manually:
